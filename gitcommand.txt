git rm filename -> delete a file completly

git add filename -> to save changes 
git commit -m 'author ' -> to save commit 
git log --oneline -> to hissory of commited 
git checkout idOfcommit -> to see files before commiting this git id 
git revert idOfcommit -> to  delete the commit of respective git id

git checkout -f -> to restore the last file before change 
git branch -> to create a new  branch
